Emacs modes for editing Itasca software data files.
---------------------------------------------------
FLAC FLAC3D UDEC 3DEC PFC www.itascacg.com/software

Installation: put this code into your .emacs file


to set the mode on a per-file basis:
-*- mode: itasca-general -*-
-*- mode: itasca-flac -*-
-*- mode: itasca-pfc -*-
